========================================================
   RF SIGNAL MAPPER - DATA FOLDER
========================================================

Esta pasta armazena os mapas de sinal RF gerados pela
app "RF Signal Mapper Pro" do EAGLE-FIRMWARE.

FORMATO DOS FICHEIROS:
----------------------
map_YYYYMMDD_HHMMSS.txt

Exemplo: map_20251028_143025.txt

CONTEUDO:
---------
# EAGLE RF Signal Map
# Frequency: 433920000 Hz
# Points: 150
# Latitude,Longitude,RSSI,Timestamp
40.712776,-74.005974,-67,1635789456
40.712800,-74.006000,-72,1635789460
...

COMO USAR:
----------
1. Abre a app "RF Signal Mapper" no PortaPack
2. Define a frequencia alvo
3. Press START para comecar a gravar
4. Move-te pela area (com GPS se disponivel)
5. Press SAVE para exportar o mapa
6. Ficheiro e gravado nesta pasta

IMPORTAR MAPAS:
---------------
- Press LOAD na app
- Seleciona o ficheiro
- Visualiza o heatmap anterior
